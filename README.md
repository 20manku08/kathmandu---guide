# Kathmandu Travel Portal Site

This is a virtual hub built to help tourists easily navigate and plan their trips around the Kathmandu Valley. I wanted to create a clean, accessible web application that mainly focuses on local travel spots, transit routes, and cultural hubs in my town. 

This project was built as part of the Macondo 14-hour Build Campaign.

### Tech Stack:
* HTML5 / CSS3
* JavaScript
* Netlify (for hosting)

### How to Run This Project Locally:
1. Download or clone this repository to your computer.
2. Open the project folder.
